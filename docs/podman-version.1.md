## NAME
podman\-version - Display the PODMAN Version Information

## SYNOPSIS
**podman version**
[**--help**|**-h**]

## DESCRIPTION
Shows the the following information: Version, Go Version, Git Commit, Build Time,
OS, and Architecture.

**podman [GLOBAL OPTIONS]**

**podman version**
